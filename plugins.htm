Плагины (модули)

Вот список всех доступных плагинов для SVG.js с краткими пояснениями. Если вы написали что-то ещё, пожалуйста, дайте нам знать!

  ●   svg.colorat.js
svg.colorat.js - это расширение для библиотеки SVG.js, которое позволяет выбрать цвет градиента в определенной заданной позиции.
  ●   svg.connectable.js
svg.connectable.js to connect elements.
svg.connectable.js fork to connect elements (added: curved connectors, you can use any self-made path as a connector, choosable 'center'/'perifery' attachment, 'perifery' attachment for source / target SVG Paths uses smallest-distance algorithm between PathArray points)
  ●   svg.declarative.js
svg.declartive.js a plugin for SVG.js which allows for super fast, interruptible, declarative animations. All animations are physically driven, so you can get very realistic animations with this library; whilst animating a huge number of items.
  ●   svg.draggable.js
svg.draggable.js to make elements draggable.
  ●   svg.draw.js
svg.draw.js to draw elements with your mouse.
  ●   svg.easing.js
svg.easing.js for more easing methods on animations.
  ●   svg.filter.js
svg.filter.js adding svg filters to elements.
  ●   svg.foreignobject.js
svg.foreignobject.js foreignObject implementation (by john-memloom).
  ●   svg.intersections.js
svg.intersections.js find intersections of paths and lines (by amatiash)
  ●   svg.math.js
svg.math.js a math extension (by Nils Lagerkvist).
  ●   svg.panzoom.js
svg.panzoom.js a plugin for svg.js which enables panzoom for viewbox elements.
  ●   svg.path.js
svg.path.js for manually drawing paths (by Nils Lagerkvist).
  ●   svg.pathmorphing.js
svg.pathmorphing.js to make path animatable
  ●   svg.resize.js
svg.resize.js to resize elements with your mouse.
  ●   svg.screenbbox.js
svg.screenbbox.js to get the bbox in screen coordinates from transformed path/polygon/polyline
  ●   svg.select.js
svg.select.js to select elements.
  ●   svg.shapes.js
svg.shapes.js for more polygon based shapes.
  ●   svg.textmorph.js
svg.textmorph.js to make text animatable.
  ●   svg.topath.js
svg.topath.js to convert any other shape to a path.
  ●   svg.topoly.js
svg.topoly.js to convert a path to polygon or polyline.
