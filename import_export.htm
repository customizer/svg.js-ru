Импорт/Экспорт SVG

Экспорт полностью сгенерированного SVG или его части может быть выполнен с помощью метода svg(). Импорт выполняется тем же самым методом с передачей в качестве первого аргумента строкового значения svg.


svg()          как получатель (getter)
возвращает - строковое значение (string)
Экспорт полностью сгенерированного SVG или его части может быть выполнен с помощью метода svg(). 
// javascript
draw.svg()
Экспорт также работает и по отдельным элементам: 
// javascript
var rect = draw.rect()
var svg  = rect.svg()

svg()          как установщик (сеттер)
возвращает - самого себя (itself)
Импорт выполняется тем же самым методом с передачей в качестве первого аргумента строкового значения svg: 
// javascript
draw.svg('<g><rect width="100" height="50" fill="#f06"></rect></g>')
Импорт работает с любым элементом, который наследуется от SVG.Parent, то есть по сути, с любым элементом, который может содержать другие элементы.
