Установка

Различные методы установки, чтобы начать работу с SVG.js, предпочитаете ли вы менеджеры пакетов, CDN или просто включаете источник JavaScript в свой HTML.

Bower

bower install svg.js

Node

npm install svg.js

CDN.js

Возьмите последнюю версию с cdnjs.com.

Загрузка

svg.js [для разработки] - много комментариев
svg.min.js [для работы] - ± 16k при сжатии
